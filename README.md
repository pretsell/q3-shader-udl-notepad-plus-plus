# Quake 3 .shader file user defined language for Notepad++
Used for syntax highlighting when working with Quake 3 based .shader files.

## How to install
1. Download the q3.shader.xml file.
2. Copy or Cut it from your download folder / location.
3. Launch Notepad++.
4. On the top toolbar click on 'Language' > 'User Defined Language' > 'Open User Defined Language folder...'.
5. Paste q3.sahder.xml into the folder that opened.
6. Close Notepad++.
7. Launch Notepad++.
8. Open a .shader file.
9. On the top toolbar click on 'Language' > 'Q3.shader'.
10. The syntax of the .shader file should now be highlighted.
